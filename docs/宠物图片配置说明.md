# 宠物图片配置说明

## 步骤一：放置图片

将你下载的巴纳比图片**复制或移动**到项目目录：

```
petconnect-app/public/pets/barnaby.png
```

（若为 jpg 格式，命名为 `barnaby.jpg`，并在步骤三的 SQL 中相应修改）

## 步骤二：更新数据库

在 **Supabase Dashboard** → **SQL Editor** 中执行：

```sql
UPDATE public.pets
SET image_url = '/pets/barnaby.png'
WHERE id = 'barnaby';
```

若使用 jpg，将 `barnaby.png` 改为 `barnaby.jpg`。

## 步骤三：验证

1. 本地运行：`npm run dev`
2. 打开首页，巴纳比应显示新图片
3. 若仍为占位图，检查：
   - 图片是否在 `public/pets/barnaby.png`
   - 数据库是否已执行上述 SQL
   - 浏览器控制台是否有 404 报错
