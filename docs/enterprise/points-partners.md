# 积分公益合作伙伴

专属积分公益通道帮助社区救助站获得长期物资与运营支持，用户可在成长任务页直接发起捐赠，后台会写入 `points_transactions` 与 `points_donations`，方便运营对账。

## 合作伙伴 & 价值

| 合作机构 | 捐赠方案 | 说明 |
| --- | --- | --- |
| 暖心救助厨房（rescue-kitchen） | 100 分兑换一份热餐与营养守护包 | 每批次提供 50 份热餐，投递至流浪猫狗集中点和流动救助站，并通过公众号公布善款流向。 |
| 流浪医疗站（health-station） | 180 分援助一次疫苗/复诊 | 协调 6 家合作诊所，优先提供疫苗、绝育及复诊名额，积分捐赠即刻转成救助基金。 |
| 社区志愿奖助（community-hero） | 60 分用于志愿物资或勋章 | 用于购买志愿者日常物资（手套、防蚊贴、饮水），并支持社区分享会的奖励机制。 |

## 捐赠流程说明

1. 客户端调用 Supabase RPC `donate_points(p_partner_key, p_points, p_note)`，函数会在秘钥账户中扣减积分并同步写入 `points_transactions` 与 `points_donations`。
2. 后台可通过 `points_donations` 表查看每笔捐赠的机构、积分、备注与创建时间；同时可参考 `points_transactions` 展示流水明细。
3. 若需追踪公益成果，可结合 `partner_key` 与运营文案，定期向捐赠者推送对应回报（例如提供回访照片、感言）。

## 运营建议

- 每周固定发布“积分公益榜”，公布本期捐赠次数最多的 TOP3 志愿者并邀请其参与线下活动。
- 结合 `points_rank` 页面宣传公益贡献，将“救援任务+捐赠奖励”作为等级晋升的加分项。
- 提供 `note` 字段用于记录捐赠用途（如“XX 流浪狗病例援助”），便于后续合规与回馈。
